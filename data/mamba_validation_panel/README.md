# Mamba Validation Panel Spectra

Validation split derived from `Act_of_God_Mamba_Results/data/validation_panel.csv`. The panel captures labelled spectra and metadata used to evaluate the Mamba-SSM model outside the training distribution.

## Contents
- `validation_panel.csv` — consolidated spectra measurements, target concentrations, and panel annotations.
- `validation_panel_summary.md` (source) — narrative summary of the validation protocol (kept in block data directory).

Consult `schema.json` for field definitions and `provenance.yaml` for extraction details.
