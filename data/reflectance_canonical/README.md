# Reflectance Canonical Dose Aggregates

Canonical, analysis-ready tables generated by `Reflectance/reflectance/build_canonical_dataset.py`. The dataset aligns diode-array (DAD) concentrations with reflectance-derived summaries per dose and treatment.

## Contents
- `dose_summary.csv` — trimmed mean concentration estimates for each UV dose combination.
- `dose_reflectance_stats.csv` — per-dose reflectance descriptive statistics across wavelengths.
- `precision_weighted_concentrations.csv` — per-sample precision-weighted concentrations with latent residual components.
- `precision_weighted_concentrations_treatment.csv` — treatment-level aggregation of precision-weighted estimates.
- `reflectance_trimmed_stats.csv` — wavelength-level trimmed statistics for reflectance bands.
- `manifest.json` — source manifest captured by the canonical dataset build step.

Refer to `schema.json` for column-level details and `provenance.yaml` for the generation history.
