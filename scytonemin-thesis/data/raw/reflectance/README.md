# Reflectance — Raw Staging Assets

Raw and intermediary reflectance spectra copied from `UPLC_DAD_Scytonemin_Concentrations/Reflectance`. These directories back the canonical dataset builder and downstream aggregation scripts in `src/reflectance/`.

## Contents
- `denoised_full_run_staging/` — denoised spectra grouped by treatment and orientation.
- `from_dose_concentration_chromo_dad_work_upstream/` — upstream drops linking concentration pipelines with reflectance processing.

Do not edit files in place; regenerated outputs should be written to `data/reference/reflectance/` and `scaffold/reflectance/`.
